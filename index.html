Versioning

Semantic Versioning

http://semver.org

MAJOR.MINOR.PATCH

  1. MAJOR - incompatible API changes
  2. MINOR - backwards-compatible + new functionality
  3. PATCH - backwards-compatible + bug fixes

Bower

http://bower.io

Bower is a package manager for client side technologies. It can be used to search , install, uninstall web assets like JavaScript, HTML, and CSS. It is not an opinionated tool and leaves lot of choice to the developers who are using the technology.

bower.json

Manifest file similar to Nodeâ€™s package.json


  {
    "name": "my-project",
    "version": "1.0.0",
    "main": "path/to/main.css",
    "ignore": [
      ".jshintrc",
      "**/*.txt"
    ],
    "dependencies": {
      "<name>": "<version>",
      "<name>": "<folder>",
      "<name>": "<package>"
    },
    "devDependencies": {
      "<test-framework-name>": "<version>"
    }
  }
